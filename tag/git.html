Cloudflareの100MB upload size limit突破のためGit LFS→Git+DVC+Minioに移行した

Posted on 金 26 4月 2024 in 研究 • Tagged with docker, cloudflare, minio, dvc, git, server

研究データを自宅サーバーに保管し、バイナリファイルはGit LFSを使用して管理していた。しかし、Cloudflareを導入することでGit LFSでも100MB以上のファイルをアップロードすることができなくなった。そのため、チャンクアップロードに対応したMinioとバイナリファイルバージョン管理ができるDVCを使うことで100MB以上のファイルをアップロードできるようになった。
